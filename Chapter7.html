Chapter 7: Using the DOM - Summary

1. The DOM Tree and Node Types

  • The DOM (Document Object Model) represents an HTML document as a tree of nodes.
  • The root of the tree is the <html> element, and every element, text, and attribute is a node.
  • Node types include:
    • Element nodes: HTML tags like <p> or <img>
    • Text nodes: The text inside elements
    • Attribute nodes: Attributes such as src or id
    • Comment nodes: Represent <!-- comments -->

2. Node Lists

  • document.querySelectorAll() selects multiple elements and returns a NodeList.
  • Example:
    document.querySelectorAll("p")[1]; — selects the second <p> element.

3. Adding New Nodes

  • You can create and add new elements dynamically to a webpage.
  • Steps:
    1. createTextNode() — create a text node.
    2. createElement() — create an element.
    3. appendChild() — attach the text to the element.
    4. appendChild() again — add the new element to the document.
  • Example:
    let newP = document.createElement("p");
    let textNode = document.createTextNode("Typescript");
    newP.appendChild(textNode);
    document.body.appendChild(newP);

4. Example: Shopping Cart

  • When a button is clicked, the book title is added to the order list.
  • Example:
    orderButton.addEventListener("click", () => {
      let newP = document.createElement("p");
      let textNode = document.createTextNode(title.innerText);
      newP.appendChild(textNode);
      orderInfo.appendChild(newP);
    });
  • Style new elements using style properties, e.g. newP.style.color = "blue";

5. Adding Nodes with Attributes

  • To create elements like <img> that need attributes:
    let newImg = document.createElement("img");
    newImg.setAttribute("src", "images/wall.jpg");
    document.body.appendChild(newImg);

6. Inserting Before Existing Nodes

  • Use insertBefore(newNode, referenceNode) to add elements before others.
  • Example:
    let tsNode = document.createElement("p");
    let tsText = document.createTextNode("Typescript");
    tsNode.appendChild(tsText);
    let basisNode = document.querySelectorAll("p")[0];
    document.body.insertBefore(tsNode, basisNode);

7. Deleting Nodes

  • remove() — directly removes an element.
    Example: document.querySelector("h1").remove();
  • removeChild() — removes a specific child from a parent.
    Example: this.parentNode.removeChild(this);

8. Example: Click to Delete

  • Clicking an item removes it from the list:
    const items = document.querySelectorAll("li");
    for (let item of items) {
      item.addEventListener("click", function() {
        this.parentNode.removeChild(this);
      });
    }
  • In function(){}, this refers to the clicked element.

9. Example: Delete Buttons

  • Each list item has a delete button (×) to remove itself.
  • Example:
    const buttons = document.querySelectorAll("p > span");
    for (let button of buttons) {
      button.addEventListener("click", function() {
        this.parentNode.parentNode.removeChild(this.parentNode);
      });
    }

10. Project: My Book List App

  • A small app to add and delete books dynamically.
  • Example:
    save.addEventListener("click", (e) => {
      e.preventDefault();
      const item = document.createElement("li");
      item.innerHTML = `${title.value} - ${author.value} <span id="delButton">Delete</span>`;
      bookList.appendChild(item);
      title.value = ""; author.value = "";
    });

11. Summary

  • The DOM allows JavaScript to dynamically add, insert, and remove HTML elements.
  • Core methods: createElement(), appendChild(), insertBefore(), remove().
  • The this keyword inside functions refers to the element that triggered the event.
  • These methods form the foundation for interactive web applications like shopping carts and book lists.
