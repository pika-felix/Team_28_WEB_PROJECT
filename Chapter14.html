Chapter 14: Drawing with Canvas - Summary

1. What is Canvas?

  • The <canvas> element allows drawing graphics directly on a web page using JavaScript.
  • Canvas uses the Canvas API to draw shapes, images, and animations.
  • It is commonly used for games, drawing tools, and interactive visual effects.

2. Characteristics of Canvas

  • Canvas works at the pixel level, giving detailed control over graphics.
  • It is suitable for animations and real-time rendering.
  • Canvas graphics are not part of the DOM after being drawn.

3. Creating a Canvas

  • The canvas area is defined using the <canvas> tag.
  • Width and height should be set using attributes, not CSS.
  • The drawing area can be resized to fit the browser window.

4. Rendering Context

  • A canvas must use a rendering context to draw graphics.
  • The 2D context is created using:
    canvas.getContext("2d")
  • The context provides drawing methods and styles.

5. Coordinate System

  • The top-left corner of the canvas is (0, 0).
  • The x-axis increases to the right, and the y-axis increases downward.
  • Angles are measured in radians.

6. Drawing Shapes

  • Rectangles can be drawn using fillRect() and strokeRect().
  • Paths are used to draw lines, triangles, and custom shapes.
  • Circles and arcs are drawn using the arc() method.

7. Paths and Curves

  • Paths are created using methods like beginPath() and closePath().
  • BÃ©zier curves allow smooth and natural curves.
  • The Path2D object allows reusing complex shapes.

8. Images and Clipping

  • Images can be drawn on canvas using drawImage().
  • Images must load completely before being drawn.
  • Clipping restricts drawing to a specific shape.

9. Transformations

  • Canvas supports scaling, rotation, and translation.
  • Transformations affect how shapes and images are rendered.

10. Summary

  • Canvas enables powerful graphical drawing using JavaScript.
  • It is ideal for animations, games, and interactive graphics.
  • Understanding paths, shapes, images, and transformations is essential.
