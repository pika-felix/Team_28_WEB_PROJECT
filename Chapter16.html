Chapter 16: Canvas Animation - Summary

1. Drawing Shapes Using Objects

  • Canvas animations often use objects to represent shapes.
  • A constructor function is used to create reusable shapes like circles.
  • Each object stores properties such as position, radius, and color.
  • A draw() method renders the shape using Canvas methods.

2. Creating Multiple Objects

  • Loops are used to create many shapes efficiently.
  • Random values generate different positions, sizes, and colors.
  • Objects are stored in an array for easy management.
  • Reloading the page creates a new random animation layout.

3. Basic Animation Concept

  • Animation is created by updating an objectâ€™s position repeatedly.
  • Movement is controlled using speed values such as dx and dy.
  • The object is redrawn at a new position in each frame.

4. Using requestAnimationFrame()

  • requestAnimationFrame() creates smooth animations.
  • It repeatedly calls a function to update and redraw the canvas.
  • This method is more efficient than traditional timers.

5. Clearing and Redrawing the Canvas

  • The canvas must be cleared before drawing each new frame.
  • clearRect() removes previous drawings.
  • This prevents visual overlap and creates smooth motion.

6. Bouncing Animation

  • Collision detection checks if an object hits the canvas edges.
  • When an edge is reached, the direction is reversed.
  • This creates a bouncing effect.

7. Animating Multiple Objects

  • Each object has its own movement and animation logic.
  • All objects are animated inside a loop.
  • Multiple objects move independently at the same time.

8. Summary

  • Canvas animation is based on updating positions and redrawing frames.
  • requestAnimationFrame() is essential for smooth animation.
  • Objects and arrays make animations scalable and organized.
  • These techniques are used in games and interactive graphics.
