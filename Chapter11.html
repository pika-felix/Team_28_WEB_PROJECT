Chapter 11: Arrays and Objects - Summary

1. Default Parameters

  • Functions can assign default values to parameters when no argument is provided.
  • Useful for avoiding errors and simplifying function calls.

2. Spread and Rest Operators

  • Rest operator (...) collects multiple arguments into an array.
  • Spread operator (...) expands arrays into individual elements.
  • Used for copying arrays, merging arrays, and handling flexible parameters.

3. Object Properties

  • Bracket notation is required for keys with spaces or special characters.
  • Computed property names allow dynamic keys using expressions.
  • Shorthand properties simplify object creation when variable names match keys.

4. Symbols

  • Symbols are unique and create hidden, private-like object properties.
  • Symbol keys must always use bracket notation.
  • Symbol.for() creates or retrieves global shared symbols.

5. Destructuring Assignment

  • Extracts values from arrays and objects into variables easily.
  • Supports renaming, default values, and nested destructuring.
  • Useful for cleaner and more readable code.

6. Array Methods: map, filter, reduce

  • map() transforms each element and returns a new array.
  • filter() returns elements that meet a condition.
  • reduce() processes all elements to produce a single value.

7. Map Object

  • Stores key-value pairs where keys can be any type (objects included).
  • Preserves insertion order.
  • Provides methods like set(), get(), has(), and clear().

8. Set Object

  • Stores unique values without duplicates.
  • Useful for removing duplicates or performing membership checks.

9. Iterable and Iterator Concepts

  • Arrays, strings, maps, and sets are iterables.
  • They support for...of, spread syntax, and destructuring.
  • Use Symbol.iterator to create custom iterators.

10. Generator Functions

  • Defined using function* and produce values using yield.
  • Generators pause and resume execution.
  • Useful for custom iteration and asynchronous workflows.

11. Summary

  • Arrays and objects are essential for storing and managing data.
  • Modern ES6 features (spread, rest, destructuring, symbols) improve flexibility and readability.
  • Map and Set offer more powerful alternatives to plain objects and arrays.
  • Generators and iterators provide advanced control over data flow.
