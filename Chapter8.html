Chapter 8: Built-in Objects - Summary

1. What Are Built-in Objects?

  • Built-in objects are predefined JavaScript objects available for direct use.
  • When a web page loads, the window object is created first, followed by other browser-related objects.
  • These objects are part of the HTML Web API, used to manage browser behavior and web page interaction.

2. Window Object

  • The window object is the top-level object of the browser.
  • It controls the browser window and includes all other objects.
  • Methods:
    • alert(), prompt(), confirm() — show dialogs.
    • open(), close() — open or close browser windows.
  • Example:
    window.open("notice.html", "popup", "width=600,height=500,left=300,top=200");

3. Screen Object

  • Provides information about the user's screen size and available area.
  • Example: Centering a popup window
    let left = (screen.availWidth - popWidth) / 2;
    let top = (screen.availHeight - popHeight) / 2;
    window.open("notice.html", "event", `width=600,height=500,left=${left},top=${top}`);

4. History Object

  • Contains the user's browsing history.
  • Used for navigation, such as moving forward or backward.
  • Example:
    history.back(); // Go to previous page
    history.forward(); // Go to next page

5. Location Object

  • Stores information about the current page URL, including protocol, hostname, and path.
  • Example:
    location.href = "https://www.example.com"; // Redirects to a new page

6. Date Object

  • Used for managing and displaying dates and times.
  • Creating Date objects:
    • new Date() — current date/time
    • new Date("2025-02-25") — specific date
    • new Date("2025-02-25T18:00:00") — date and time
  • Main methods:
    • Get: getFullYear(), getMonth(), getDate(), getDay()
    • Set: setFullYear(), setMonth(), setDate()
    • Display: toLocaleString(), toDateString(), toTimeString()

7. Example: Counting Days Passed

  • Calculates the number of days between two dates.
  • Example:
    const firstDay = new Date("2021-07-01");
    const today = new Date();
    const passed = today.getTime() - firstDay.getTime();
    const days = Math.round(passed / (1000 * 60 * 60 * 24));
    document.querySelector("#result").innerText = days;

8. Example: Digital Clock

  • Displays current time on the webpage and updates every second.
  • Example:
    function clock() {
      let now = new Date();
      let hrs = now.getHours();
      let mins = now.getMinutes();
      let secs = now.getSeconds();
      let period = hrs >= 12 ? "PM" : "AM";
      if (hrs > 12) hrs -= 12;
      if (hrs === 0) hrs = 12;
      hrs = hrs < 10 ? "0" + hrs : hrs;
      mins = mins < 10 ? "0" + mins : mins;
      secs = secs < 10 ? "0" + secs : secs;
      document.querySelector("#clock").innerText = `${period} ${hrs}:${mins}:${secs}`;
    }
    setInterval(clock, 1000);

9. Math Object

  • Provides mathematical constants and functions.
  • Examples:
    • Math.PI — π constant
    • Math.round(), Math.ceil(), Math.floor() — rounding numbers
    • Math.random() — random number between 0 and 1
  • Example: Random number between 1–100
    Math.floor(Math.random() * 100 + 1);
