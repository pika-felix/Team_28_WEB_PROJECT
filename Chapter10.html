Chapter 10: Strings and Arrays - Summary

1. Understanding Strings

  • Strings are primitive data types, but JavaScript temporarily converts them into String objects when using properties or methods (autoboxing).
  • The length property shows how many characters a string has.
  • Characters can be accessed using charAt() or bracket notation (str[0]).
  • Useful methods include indexOf(), includes(), startsWith(), endsWith(), trim(), and case-changing methods.
  • Substring extraction uses slice() or substring().
  • split() converts a string into an array.

2. Using Regular Expressions

  • Regular expressions describe text patterns and help search or validate strings.
  • test() checks if a pattern exists; match() extracts matching parts.
  • Patterns can include digits, letters, repetitions, anchors (^ and $), and flags like i for case-insensitive matching.
  • Examples include matching student IDs, phone numbers, or specific words.

3. Converting Strings and Arrays

  • Strings are hard to modify directly, so they are often converted to arrays.
  • Conversion methods include split(), [...str] (spread syntax), and Array.from().
  • Arrays can be converted back to strings using join().

4. Introduction to Arrays

  • Arrays can be created using literals ([]), constructors (new Array()), or assigned values directly.
  • Elements can be accessed and modified using indexes.
  • Empty slots can exist if indexes are skipped.

5. Looping Through Arrays

  • for loops allow index-based iteration.
  • forEach() runs a callback for every element and can use value, index, and the array itself as parameters.
  • forEach() cannot use break or continue.

6. Combining and Copying Arrays

  • concat() merges arrays or values into a new array.
  • The spread operator (...) is a modern, more flexible way to combine arrays.

7. Sorting, Reversing, and Slicing Arrays

  • reverse() flips the order of elements in place.
  • sort() organizes elements but sorts numbers incorrectly unless given a compare function ((a, b) => a - b).
  • slice() creates a new copy of a portion of the array without modifying the original.

8. Adding and Removing Elements

  • push() adds to the end; pop() removes from the end.
  • unshift() adds to the beginning; shift() removes from the beginning.
  • splice() removes or replaces elements at specific positions and modifies the original array.

9. Overall Summary

  • Strings and arrays are essential data structures used for text and list processing.
  • Strings rely on methods for searching, slicing, and formatting.
  • Arrays offer powerful tools for sorting, looping, and modifying data.
  • Understanding conversions between strings and arrays is important for modern JavaScript development.
