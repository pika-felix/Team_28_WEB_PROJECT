Chapter 5: Basics of the DOM - Summary

1. Understanding the DOM

  • DOM (Document Object Model) is a structured representation of an HTML document using objects.
  • It allows JavaScript to access, modify, and control web page elements dynamically.
  • The DOM organizes HTML into a hierarchical tree structure called the DOM tree.
  • Each element, attribute, and text node can be accessed and changed using JavaScript.

2. The DOM Tree Structure

  • Represents the parent–child relationships between elements.
  • The <html> element is the root node, with <head> and <body> as children.
  • Each node in the tree can represent an element, its text, or its attributes.

3. Accessing Web Elements

  • Use document.querySelector() to select a single element.
  • Use document.querySelectorAll() to select multiple elements (returns a node list).
  • Examples:
    • document.querySelector("h1")
    • document.querySelector("#profile")
    • document.querySelectorAll(".user")
  • Older methods: getElementById(), getElementsByClassName(), getElementsByTagName().

4. Getting and Changing Content

  • To get or modify an element’s content:
    • innerText — gets/sets visible text only.
    • innerHTML — includes HTML tags.
    • textContent — retrieves all text including hidden or extra spaces.
  • Example:
    document.querySelector("#desc").innerText;

5. Handling Click Events

  • You can make pages interactive by responding to user clicks.
  • Example:
    const title = document.querySelector("#title");
    title.onclick = () => title.innerText = "Profile";
  • Images can also be changed dynamically:
    pfImage.onclick = () => pfImage.src = "images/pf2.png";

6. Modifying Styles with JavaScript

  • Access and change CSS properties using element.style.
  • Examples:
    title.style.color = "blue";
    title.style.backgroundColor = "yellow";
  • Use camelCase for multi-word CSS properties (e.g., fontSize instead of font-size).

7. Working with Class Lists

  • classList helps manage CSS classes applied to an element.
  • Common methods:
    • add() — add a class
    • remove() — remove a class
    • contains() — check if a class exists
    • toggle() — add/remove a class automatically
  • Example:
    title.onclick = () => title.classList.toggle("clicked");

8. Accessing and Handling Forms

  • Form elements can be accessed using id, name, or document.forms.
  • Examples:
    • document.querySelector("#order-name").value;
    • document.order.product.value;
    • document.forms[0].elements[4];

9. Working with Select Elements

  • To access a dropdown list:
    const selectMenu = document.querySelector("#major");
  • Get the selected option:
    selectMenu.options[selectMenu.selectedIndex].innerText;
  • Example:
    alert(`[${selectedText}] selected.`);

10. Radio Buttons and Checkboxes

  • Grouped using the same name attribute and accessed as a list.
  • To get the selected radio button:
    document.querySelector("input[name='subject']:checked");
  • To get selected checkboxes:
    document.querySelectorAll("input[name='mailing']:checked");
  • Use the checked property to check whether a box is selected.

11. Summary

  • The DOM provides a structured way to interact with web documents.
  • JavaScript can access, edit, and style elements through the DOM tree.
  • Event handling and class manipulation enable dynamic web behavior.
  • Form, select, and input handling are key for user interaction.
