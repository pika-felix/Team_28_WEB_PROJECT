Chapter 15: Handling Graphic Elements with Canvas - Summary

1. Styling Shapes and Colors

  • Canvas uses fillStyle to set fill colors and strokeStyle to set line colors.
  • Colors can be defined using names, hex values, RGB/RGBA, or HSL/HSLA formats.
  • Transparency is controlled with globalAlpha or RGBA color values.

2. Gradients

  • Linear gradients change colors in a straight direction using createLinearGradient().
  • Radial gradients spread colors outward from a center using createRadialGradient().
  • Multiple colors are added with addColorStop().

3. Pattern Fills

  • Images can be used as repeating patterns with createPattern().
  • Pattern options include repeat, repeat-x, repeat-y, and no-repeat.

4. Shadow Effects

  • Shadows are applied using shadowColor, shadowBlur, shadowOffsetX, and shadowOffsetY.
  • Shadow settings must be defined before drawing the shape.

5. Line Styles

  • Line thickness is controlled with lineWidth.
  • Line endings and joins are styled using lineCap and lineJoin.
  • miterLimit adjusts sharp corner behavior.

6. Interactive Drawing with Mouse Events

  • Mouse events such as mousedown, mousemove, and mouseup enable drawing on the canvas.
  • Users can change colors and line width dynamically.
  • The canvas can be cleared using clearRect().

7. Transforming Graphics

  • Transformations affect the canvas context, not individual shapes.
  • translate(), rotate(), and scale() control movement, rotation, and size.
  • save() and restore() help manage transformation states.

8. Compositing Graphics

  • globalCompositeOperation defines how overlapping drawings combine.
  • Modes include source-over, source-in, destination-over, lighter, and xor.
  • These modes create advanced visual effects.

9. Summary

  • Canvas provides powerful tools for drawing and styling graphics.
  • Developers can create interactive and visually rich applications.
  • Understanding transformations and compositing is essential for advanced graphics.
